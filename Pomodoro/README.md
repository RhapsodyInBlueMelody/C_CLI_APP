## Installation

```
git clone https://github.com/RhapsodyInBlueMelody/C_CLI_APP.git 

cd C_CLI_APP/Pomodoro

./install.sh
```


## Usage

```
  pomodoro <workminute> <breakminute>
```

## Example
```
 ./pomodoro 25 5
25 minute work and 5 minutes of break
```
